# signum

A new C++ audio synthesis engine.

## Installation

### OS X

To build with Homebrew dependencies:

```
brew install libsndfile libsoundio gsl
./waf
```

### Linux (Ubuntu 14)

* Install [libsoundio](http://libsound.io/)
* `apt-get install libgsl0-dev libsndfile1-dev`
* `./waf`
