# RandomImpulse

Generate random impulses at the given frequency, with either uniform or poisson distribution.

Signature:
```python
RandomImpulse(frequency=1.0, distribution=SIGNALFLOW_EVENT_DISTRIBUTION_UNIFORM, reset=None)
```
