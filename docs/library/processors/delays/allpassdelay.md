# AllpassDelay

All-pass delay, with feedback between 0 and 1. delay_time must be less than or equal to max_delay_time.

Signature:
```python
AllpassDelay(input=0.0, delay_time=0.1, feedback=0.5, max_delay_time=0.5)
```
