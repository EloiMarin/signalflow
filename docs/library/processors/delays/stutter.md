# Stutter

Stutters the input whenever a signal is received on clock. Generates stutter_count repeats, with duration stutter_time.

Signature:
```python
Stutter(input=0.0, stutter_time=0.1, stutter_count=1, clock=None, max_stutter_time=1.0)
```
