# SawOscillator

Produces a (non-band-limited) sawtooth wave, with the given `frequency` and `phase` offset. When a `reset` or trigger is received, resets the phase to zero.

Signature:
```python
SawOscillator(frequency=440, phase=None, reset=None)
```
