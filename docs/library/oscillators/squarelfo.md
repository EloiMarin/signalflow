# SquareLFO

Produces a pulse wave LFO with the given frequency and pulse width, ranging from min to max, where width=0.5 is a square wave.

Signature:
```python
SquareLFO(frequency=1.0, min=0.0, max=1.0, width=0.5, phase=0.0)
```
